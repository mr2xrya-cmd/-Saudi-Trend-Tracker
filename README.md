# 🚀 نظام تتبع الترند السعودي والربط مع مخازن (مجاني للأبد)

هذا النظام هو مساعدك الذكي الذي يعمل آلياً لجمع أفضل 20 منتج ترند يومياً من (تيك توك، جوجل، سناب، انستقرام، يوتيوب، X) ويقوم بتحليلها بذكاء اصطناعي (Gemini Pro) وربطها بمتجر "مخازن" لإرسال تقرير شامل لك عبر تيليجرام.

## ✨ الميزات الرئيسية:
- **تتبع الترند:** مسح شامل لكافة منصات التواصل الاجتماعي.
- **التحليل الذكي:** وصف تسويقي، سكريبت إعلان، ونقاط الألم باستخدام Gemini Pro.
- **ربط مخازن:** جلب الأسعار، الروابط، وتوفر المخزون آلياً.
- **رادار المناسبات:** ربط المنتجات بالرواتب والمواسم السعودية.
- **مجاني تماماً:** يعمل عبر GitHub Actions دون أي تكاليف استضافة.

## 🛠️ كيفية الإعداد (في 3 دقائق):
1. قم بإنشاء مستودع (Repository) جديد في حسابك على GitHub (مجاني).
2. ارفع كافة الملفات الموجودة في هذا المجلد إلى المستودع.
3. اذهب إلى إعدادات المستودع (Settings) -> Secrets and variables -> Actions.
4. أضف الـ Secrets التالية:
   - `GEMINI_API_KEY`: مفتاح Gemini Pro الخاص بك.
   - `TELEGRAM_BOT_TOKEN`: توكن بوت التيليجرام.
   - `TELEGRAM_CHAT_ID`: معرف حسابك في تيليجرام.
   - `MAKHAZEN_USER`: بريدك الإلكتروني في مخازن.
   - `MAKHAZEN_PASS`: كلمة مرور مخازن.

## 📅 متى يعمل النظام؟
النظام مبرمج ليعمل تلقائياً كل يوم في الساعة **10:00 صباحاً بتوقيت السعودية**. يمكنك أيضاً تشغيله يدوياً في أي وقت من خلال تبويب **Actions** في GitHub.
